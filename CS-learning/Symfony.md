# Problems

- ## Error message: no PHP binaries detected
  -  Command:
  ```
  $ symfony php --version
  ```
  -  How to solve:
  ```
  $ symfony local:php:refresh
  ```
  
  
# Commands
## Symfony
- ### For running the web server with TLS support
```
$ symfony server:ca:install
```
  
## Composer
- ### For updating a package
```
$ composer update <package name>
```
